Задание 1¶

Случайная величина распределена равномерно на отрезке $[0, \theta]$. По выборке объема $n$ найдены оценки параметра $\theta$:

Случайная величина: $\xi \sim R(0, \theta)$

Выборка: $\overline{X}_{n} = (x_{1}, ..., x_{n})$

$\widetilde{\theta}_{1} = 2 \overline{x}$, $\widetilde{\theta}_{2} = x_{min}$, $\widetilde{\theta}_{3} = x_{max}$, $\widetilde{\theta}_{4} = x_{min} + x_{max}$, $\widetilde{\theta}_{5} = x_{1} + \frac{1}{n-1} \sum\limits_{k=2}^n x_{k}$.

а) Проверить оценки на несмещенность и состоятельность. Исправить эти оценки, если необходимо. Какая из исправленных оценок более эффективна?

б) Сгенерируйте выборку объема $N = 100$ для некоторого значения параметра $\theta$. Для всех значений $1 \le n \le N$ вычислите исправленные оценки и постройте графики $|\widetilde{\theta}_{k} - \theta|$ в зависимости от $n$.

def theta_difference(theta_estimate, theta, X):
    diff = np.zeros(len(X))
    for i in range(0, len(X)):
        diff[i] = np.abs(theta_estimate(X[:i+1]) - theta)
    return diff
